Declaración de accesibilidad

Compromiso de NeoPulsor con una navegación clara, usable y accesible.

1. Compromiso de accesibilidad

NeoPulsor trabaja para que este sitio web sea accesible, claro y usable para el mayor número posible de personas, independientemente del dispositivo, navegador o forma de navegación utilizada.

2. Medidas aplicadas

En el desarrollo de esta web se han tenido en cuenta criterios como una estructura HTML semántica, textos alternativos en imágenes relevantes, contraste visual adecuado, navegación mediante enlaces claros y diseño adaptable a dispositivos móviles.

3. Navegación y estructura

La web está organizada en secciones identificables: inicio, sobre NeoPulsor, servicios, portfolio, clientes, reseñas y contacto. Además, las páginas legales están enlazadas desde el pie de página.

4. Compatibilidad

Este sitio está diseñado para funcionar correctamente en navegadores modernos y en diferentes tamaños de pantalla, incluyendo ordenadores, tablets y teléfonos móviles.

5. Contenido audiovisual

Al tratarse de una web de producción audiovisual, puede incluir vídeos, imágenes o recursos visuales. Se procurará que estos contenidos no dificulten la navegación ni impidan el acceso a la información principal.

6. Posibles limitaciones

Aunque se intenta mantener un buen nivel de accesibilidad, pueden existir contenidos o recursos externos que no dependan directamente de NeoPulsor, como vídeos incrustados, redes sociales o plataformas de terceros.

7. Contacto

Si encuentras alguna barrera de accesibilidad o tienes dificultades para acceder a cualquier contenido de la web, puedes escribir a: neopulsor@gmail.com.
